#  Sales Data Analysis Dashboard

##  Project Overview

Built an end-to-end sales analytics dashboard using real-world retail data.

This project analyzes sales, profit, and customer trends to generate actionable business insights.

---

##  Key Features

* Data Cleaning & Preprocessing
* Sales Analysis by Region, Category, and Time
* Interactive Dashboard using Streamlit
* KPI Metrics (Total Sales & Profit)

---

## 📊 Key Insights

* Identified top-performing regions
* Analyzed monthly revenue trends
* Found most profitable product categories

---

##  Tech Stack

* Python
* Pandas
* Matplotlib
* Streamlit
